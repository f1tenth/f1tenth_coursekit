Welcome to the official course kit documentation of F1tenth.

Here, you will find all the information needed to get your own f1tenth course running at your institution. This site only contains course materials. Information on how to build the cars and get started can be found here.

This is not a public documentation as it contains solutions to the labs.

There are three parts to this documentation.

  1. Course Overview gives a brief overview of what the class is about and a snapshot of the materials that are presented. It also details the prerequisities needed and the syllabus.
  2. Course Sessions contains lecture materials (recordings and slides) and information on the final project.
  3. Assignments has lab handouts and solutions.

For files of the f1tenth logo, see Press

Join in the discussion with us on Slack! This is a Slack workspace that is specific to educators, collaborators, and organizers.

The table of contents below and in the sidebar should let you easily access the documentation for your topic of interest. You can also use the search function in the top left corner.

Note

F1tenth is an open source project developed by a community of researchers and students. The documentation team can always use your feedback and help to improve the tutorials and class reference.
